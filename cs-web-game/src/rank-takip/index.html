Rank nasıl işler?

  • Rank yalnızca üye girişi yapan hesaplara yazılır.
  • Misafir oyuncular sunucuda oynayabilir; sıralamaya hiç yazılmaz.
  • Üye bir misafiri öldürürse kill, üye hesaba sayılır.
  • Misafir birini öldürürse misafire rank yazılmaz; kurban üyeyse ölümü hesaba işlenebilir.
  • Sıralama hesap bazlıdır — oyun içi nick değişikliği rank’ı sıfırlamaz.
